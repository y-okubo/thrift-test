# Ruby
## Server
```
cd ruby
ruby server.rb
```

## Client
```
cd ruby
ruby client.rb
```

# Java
## Server
```
cd java
mvn compile && mvn exec:java -Dexec.mainClass=Server
```

## Client
```
cd java
mvn compile && mvn exec:java -Dexec.mainClass=Client
```

# Go
## Server
```
cd go
go run ./server/*.go
```

## Client
```
cd go
go run ./client/*.go
```
